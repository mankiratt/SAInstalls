## Photo specs

- **Format:** JPG for photos, SVG for the logo
- **Quality:** Save JPGs at 80–85% quality for web
- **Max dimensions:** 1200px wide for gallery, 1920px for hero
- **Alt text:** Already written in the HTML — just replace the placeholder `<div>` with `<img src="..." alt="...">`
